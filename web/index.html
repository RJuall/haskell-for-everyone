Does the Haskell compiler or runtime environment come with Haskell For Everyone?

No. Haskell must be installed on your computer to use Haskell for Everyone. Haskell can be found on Haskell's official website.

What are rooms?

The rooms feature uses websockets to allow a group of users to perform code sharing during a session. Anyone can create a public or private room, if the room is private there must be a password. The creator of the rooms can also set whether anyone can edit or if it is read only for everyone except the owner. Anyone can join a public room and for a private room the password must be known to join. Once the room is empty, the room is removed from the server.
